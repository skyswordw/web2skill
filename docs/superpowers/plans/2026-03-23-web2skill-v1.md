# web2skill v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a standalone Python repository that exposes stable, replayable, agent-usable ModelScope skills through a validated runtime and CLI.

**Architecture:** The repository is split into strict ownership slices so docs/bootstrap, runtime/browser infrastructure, skill packaging/CLI, provider logic, and evals can evolve independently. The runtime prefers captured network data, degrades to DOM parsing only when needed, and records replayable traces for every invocation.

**Tech Stack:** Python 3.13/3.14, uv, Playwright, Typer, Pydantic v2, pytest, Ruff, Pyright
